# Portfolio Project Rules

## Project Screenshots and Image Carousels
When adding a new project to the portfolio:
1. Ensure the project data in the API/Database includes a `screenshots` array containing paths to multiple images.
2. The `ProjectCard.jsx` and `ProjectDetailsPage.jsx` components are already configured to automatically render a looping animated image carousel if the `screenshots` array is provided and contains more than 1 image.
3. Always ask the user to provide 4-5 high-quality screenshots for any new project to take full advantage of the animated carousel design.
